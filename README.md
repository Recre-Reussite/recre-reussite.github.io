# Récré & Réussite

Website for **Récré & Réussite**, a childcare services company based in France.

Built with [Astro](https://astro.build/) and [Tailwind CSS](https://tailwindcss.com/), deployed as a fully static site on GitHub Pages.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 18

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

### Build

```bash
npm run build
```

The static output is generated in `./dist`.

### Preview

```bash
npm run preview
```

## Deployment

The site is automatically built and deployed to GitHub Pages via GitHub Actions on every push to `main`.

Custom domain: [recre-reussite.fr](https://recre-reussite.fr)

## Tech Stack

| Tool | Purpose |
|------|---------|
| Astro | Static site generator |
| Tailwind CSS v4 | Utility-first styling |
| Lucide Icons | Iconography |
| Nunito | Primary typeface (Google Fonts) |
| GitHub Actions | CI/CD |
| GitHub Pages | Hosting |

## Design

The visual identity follows the brand guidelines defined in [`DESIGN.md`](./DESIGN.md). Key traits: playful, colorful, rounded shapes, warm tones — building trust with parents while conveying a joyful, child-friendly energy.
